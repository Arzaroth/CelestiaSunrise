Celestiasunrise

Savegame editor for the mobile game "My Little Pony"

Download this project as a .zip file Download this project as a tar.gz file

What is it?

Celestia Sunrise is a savegame editor for the quite popular "My little Pony" mobile game.

It provides both a graphical interface (suitable for most users) and a command-line interface (for people like who only use something that works in their shell).

It currently only works for Android devices, but I plan to introduce iOS compatibility.

Instructions

Each line starting with ">" are meant to be run in a console.

If you are a Linux user:

  • > git clone https://github.com/Arzaroth/CelestiaSunrise.git
  • > cd CelestiaSunrise
  • > python3.4 -m venv .
  • > . bin/activate
  • > pip install -r requirements.txt
  • > python CelestiaSunrise.py -g
  • Profit !

Note that I recommend you to take a look at pew for your virtualenvs management.

If you are a Windows user:

  • Download and unzip latest release. Write down where you put the uncompressed folder (the one containing a requirements.txt file).
  • Install python from here. Be sure to check the "Add anaconda to my PATH environment variable" checkbox.
  • Open a console (the "cmd" program).
  • Browse to the previously uncompressed folder using the "cd" command. e.g.:

    > cd Downloads\CelestiaSunrise

  • Run the following command:

    > conda create -n celestiasunrise python=3

  • Run the following command:

    > conda install -n celestiasunrise pip

  • Run the following command:

    > activate celestiasunrise

  • Run the following command:

    > pip install -r requirements.txt

  • Run the following command:

    > python CelestiaSunrise.py -g

  • Profit !

Note that you will have to run the "activate" command each time you want to use this tool. You find this complicated ? Take a look at this, or even better, at this.

Basic usage:

> python CelestiaSunrise.py -g

Shell-like usage:

> python CelestiaSunrise.py <save_file> <encrypt_key>
